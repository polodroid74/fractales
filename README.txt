//Procédure à suivre pour executer le code\\

1-Positioner le terminal dans le dossier "sujet1"

2-saisir la commande make

3- Saisir la commande ./koch <longeur du segment> <nombre d'itérations> <couleur du tracé> <couleur de l'arrière-plan> <nom du fichier> <all>

Remarques:

	-Une saisie érronée ou incomplète lancera automatiquement le mode intéractif.
	-Les couleurs sont à saisir en hexadécimal.
	-Le paramètre all servant à conserver toutes les images est facultatif.
	-Le ".ppm" à la fin du nom du fichier est facultatif.
	-le fichier test.adb n'est pas déstiné à être utilisé pour générer des fractales, utilisez plutôt le fichier koch.adb.
